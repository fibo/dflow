dflow

A minimal Dataflow programming engine

Features

  • Expressive and simple API
  • It is easy to create nodes.
  • Written in TypeScript. Whole implementation is in this dflow.ts file. Transpiled and minified it is around 4kb.

API

Types

DflowNode

DflowNode represents a block of code: it can have inputs and outputs. 


import { Dflow, type DflowNode } from "dflow";

const Sum: DflowNode = {
  kind: "sum",
  inputs: [Dflow.input("number"), Dflow.input("number")];
  outputs: [Dflow.output("number")];
  run: (a: number, b: number) {
    return a + b;
  }
}
